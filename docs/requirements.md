## 2. Thông Tin Đề Tài
- Tên đề tài:  Hệ Thống Quản Lý Nhà Trọ / Ký Túc Xá
- Mô tả ngắn:  Hệ thống giúp quản lý thông tin phòng trọ hoặc ký túc xá một cách hiệu quả . Người quản lý có thể quản lý phòng, người thuê, hợp đồng và thanh toán tiền phòng. Hệ thống giúp theo dõi trạng thái phòng, lịch sử thanh toán và hỗ trợ quản lý nhà trọ thuận tiện hơn so với phương pháp thủ công
́- Đối tượng sử dụng: 
 - Chủ nhà trọ / Quản lý ký túc xá 
-  Người thuê phòng ( sinh viên / người lao động )
## 3. Tính Năng Chính (MVP)
- Tính năng 1 [ Quản lý tài khoản ] :
•	Đăng ký / đăng nhập tài khoản 
•	Phân quyền người dùng (Admin/Người thuê)
- Tính năng 2 [ Quản lý phòng ] :
•	Quản lý phòng và tình trạng phòng ( thêm / sửa / xóa  )
•	Xem danh sách phòng
•	Cập nhật trạng thái phòng ( trống / đã thuê )
- Tính năng 3 [ Quản lý người thuê ] :
•	Thêm thông tin người thuê
•	Cập nhật / xóa thông tin
•	Xem danh sách người thuê


- Tính năng 4 [ Quản lý hợp đồng thuê ] :
•	Tạo hợp đồng thuê phòng
•	Theo dõi thời gian thuê
•	Xem danh sách người thuê
- Tính năng 5 [ Quản lý hóa đơn và thanh toán ] :
•	Tạo hóa đơn tiền phòng
•	Ghi nhận thanh toán
•	Xem lịch sử thanh toán
